# AI Job Intelligence Kit

一个为中文用户设计的微型内容生意 starter repo：
**把 AI 时代的新岗位、新公司、新 JD 信号，整理成可卖的认知产品。**

## 你可以卖什么

1. **电子书 / PDF**
   - 《AI时代求职信息差：100个真实岗位信号》
2. **周报订阅**
   - 每周更新最新 AI 岗位趋势、典型 JD、技能栈变化
3. **训练营配套资料**
   - 给学员做认知升级：
     - 哪些 AI 公司在招什么人
     - 哪些岗位标题其实在做什么
     - 哪些技能值得学，哪些只是噱头
4. **高客单服务前端产品**
   - 低价电子书引流 → 社群 / workshop / 1:1 咨询 / 课程

## 这个 repo 里有什么

- `docs/product-brief.zh.md`：产品定位
- `docs/ebook-outline.zh.md`：电子书大纲
- `docs/launch-checklist.zh.md`：7天上线清单
- `data/job_signal_schema.csv`：岗位信号数据库字段模板
- `samples/sample-job-signals.zh.md`：样例内容

## 最小可行产品（MVP）

先不要做“全网爬虫平台”。
先卖一个：

> **《AI求职信息差手册 Vol.1》**

建议内容结构：
- 30 家值得关注的 AI 公司
- 50 个代表性岗位 JD 拆解
- 10 类高频技能信号
- 5 个中国用户最容易误判的岗位趋势
- 一套“我该怎么补技能/投岗位”的行动框架

## 为什么这个方向可做

- 你不需要先做 SaaS，先卖内容就能验证需求
- 你可以用公开岗位信息做研究，不必先有客户案例
- 这个产品天然适合：带娃期间碎片化产出、持续滚动更新
- 一旦有人买，就能升级成更高客单产品
